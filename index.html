Bienvenidos a la presentación de tipos de archivos

A continuación se realizara una breve explicación de los siguientes temas: archivos planos, archivos xml, archivos json y por último archivos parquet.

ARCHIVO PLANO


ARCHIVO PLANO

Es un archivo de texto simple, texto sencillo o texto sin formato, es un archivo informático que contiene únicamente texto formado solo por caracteres que son legibles por humanos y carece de cualquier tipo de formato tipográfico

ARCHIVO XML


ARCHIVO XML

Los archivos XML son archivos de texto estructurados que utilizan etiquetas para definir elementos y atributos para definir sus propiedades. Estos archivos se utilizan para intercambiar información entre diferentes sistemas y aplicaciones, y se pueden utilizar para describir datos complejos y jerárquicos.

ARCHIVO JSON


ARCHIVO JSON

Los archivos tipo json (JavaScript Object Notation), es un formato el cual es implementado por Java. Hace parte de uno de los lenguajes más importantes de la programación y tiene como fin la creación de páginas web como también en la generación de múltiples software.

ARCHIVO PARQUET


ARCHIVO PARQUET

Los archivos Parquet son archivos diseñados para mejorar el rendimiento de las consultas en sistemas de procesamiento de grandes datos. Estos archivos almacenan los datos en columnas en lugar de filas, lo que permite una compresión más eficiente y una mayor velocidad de lectura. Se utilizan ampliamente en la industria para almacenar y analizar grandes cantidades de datos.
